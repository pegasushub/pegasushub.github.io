shared skills

Skills

All 7 skills are shared between Claude Code (as /pegasus-* commands) and OpenCode (as /scaffold, /debug, etc.).

scaffold

/pegasus-scaffold  ·  /scaffold

Generates a complete Pegasus workflow project from a plain-English description. Gathers 9 requirements, matches against reference patterns (tnseq, earthquake, mag, soilmoisture, airquality), and produces a workflow_generator.py, wrappers in bin/, a Dockerfile, README, and a run_manual.sh test script. Validates output against 15 checkpoints.

ScaffoldingDAG GenerationPython API
View Skill Stable

wrapper

/pegasus-wrapper  ·  /wrapper

Generates a production-ready wrapper script for a single pipeline step. Selects from 5 reference patterns — subprocess CLI, API fetch, shell with output flattening, ML training, fan-in merge — and outputs the matching Transformation Catalog and Job definitions ready to paste into your workflow generator.

Wrapper ScriptsTransformation Catalog
View Skill Stable

dockerfile

/pegasus-dockerfile  ·  /dockerfile

Generates a production-ready Dockerfile for your workflow container. Selects between python:3.8-slim, mambaorg/micromamba, and ubuntu:22.04 based on your dependency requirements. Pins versions, optimizes layer caching, sets PYTHONUNBUFFERED=1, and provides build and push commands.

DockerContainersReproducibility
View Skill Stable

convert

/pegasus-convert  ·  /convert

Converts a Snakemake or Nextflow pipeline to Pegasus. Maps every rule/process to a Transformation + Job + wrapper script, resolves wildcards into Python loops, handles support files via the Replica Catalog, and presents a side-by-side rule-to-wrapper comparison for verification.

SnakemakeNextflowMigration
